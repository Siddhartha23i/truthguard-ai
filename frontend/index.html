    Learn to Spot Misinformation

    Common Manipulation Techniques

    • Fear Mongering: Using fear without factual basis
    • False Causation: Claiming causation without evidence
    • Cherry Picking: Selecting only supporting facts
    • Context Manipulation: Wrong context for real information

    Verification Tips

    • Cross-check with multiple reliable sources
    • Look for original sources and citations
    • Check publication dates and context
    • Use reverse image search for images
    • Be skeptical of highly emotional content
